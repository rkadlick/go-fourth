# go-fourth

Personal developer portfolio built with [Astro](https://astro.build).

**Live site:** [ryanismy.name](https://ryanismy.name)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

- `src/pages/` — site routes (home, projects, blog)
- `src/content/projects/` — project entries (Markdown)
- `src/content/blog/` — blog posts (Markdown)
- `src/components/` — Astro components
- `src/styles/` — global styles and design tokens
